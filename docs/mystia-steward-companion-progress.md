# BepInEx Mod 进度摘要

更新日期：2026-06-07

## 已完成

- BepInEx Unity IL2CPP 插件入口、配置和本地 API。
- 运行时反射读取当前游戏进度、库存、流行标签、经营场景、稀客和稀客订单。
- Tauri 独立伴随窗口，默认由 Mod 自动启动。
- 伴随窗口页面：概览、普客、稀客、经营中、修改、日志。
- 夜间经营当前稀客点单推荐、专注模式、库存数量显示。
- 游戏关闭或本地 API 断开后，伴随窗口自动退出。
- 一键构建脚本和安装包打包脚本。

## 当前边界

- 不再保留独立网站功能；`pnpm build` 只用于生成伴随窗口前端产物。
- 不再读取 `.memory` 存档文件；运行时状态来自游戏内对象。
- 日志捕获可辅助识别稀客订单，但核心目标是优先依赖运行时对象。

## 后续重点

- 继续补强不同游戏版本下的运行时字段兼容。
- 减少对日志文本的兜底依赖。
- 优化伴随窗口的性能和夜间经营刷新节奏。
